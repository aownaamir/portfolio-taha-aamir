# Taha Aamir

Portfolio website for Taha Aamir, a management professional.

## 🚀 Live Demo
[View Live Project](https://taha-aamir.vercel.app/)

## 🛠 Tech Stack
React.js, Next.js, Tailwind CSS, Aceternity UI, Vercel

## ✨ Features
- Intro – Professional overview
- About Me – Background and story
- Experience – Career highlights
- Contact – Get in touch
- Socials – Links to profiles
